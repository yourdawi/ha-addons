# yourdawi Home Assistant Add-ons

This repository hosts multiple Home Assistant add-ons. Add this repository once in Home Assistant and access all add-ons listed below.


## Add-ons
- SpotConnect — virtual AirPlay/DLNA devices for Spotify Connect. See `spotconnect/README.md` for details.


## How to add this repository
1. In Home Assistant go to Settings → Add-ons → Add-on Store → Repositories.
2. Add: `https://github.com/yourdawi/ha-addons`.
3. Open the add-on you want (e.g., SpotConnect) and install it.
